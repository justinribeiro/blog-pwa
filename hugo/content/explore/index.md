I am always doing something; what that something is varies wildly based time and well, time—curiosity is never lacking in supply. This is my slightly more organized yet still ever-growing compendium of where curiosity takes me. Also, because sometimes I forget where I put things.

##

## Web Software

<p><a href="https://dof-calculator.pwa.run/">Depth of Field Calculator</a> - Because sometimes I just don't want to do the math when I'm in the field with my large format cameras. Progressive Web App (PWA).</p>

<p><a href="https://speedometer.pwa.run/">Speedometer</a> - A tiny no-frills speedometer progressive web app based on Geolocation, AmbientLightSensor, and WakeLock API. Originally developed when my speedometer on our 2005 Toyota Prius was not working. Progressive Web App (PWA).</p>

<p><a href="https://readinglight.pwa.run/">Reading Light</a> - A simple screen color changer for reading purposes. Originally developed so I could read by phone light to my twins. Progressive Web App (PWA).</p>

<p><a href="https://github.com/justinribeiro/lite-youtube">lite-youtube</a> - The fastest little YouTube embedder this side of the internet for your website. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/share-to-mastodon">share-to-mastodon</a> - A small web component that uses a dialog to allow setting of a target Mastodon instance from the user to allow easier sharing. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/code-block">code-block</a> - A web component that displays colorfully formatted code with Prism.js and LitElement. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/stl-part-viewer">stl-part-viewer</a> - A lit-element web component that uses Three.js to display an STL model file. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/toggle-tooltip">toggle-tooltip</a> - An accessible, fast, and WCAG 2.1 compliant toggle style tooltip web component using role=status. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/webnfc-reader">webnfc-reader</a> - A lit-element web component that uses WebNFC to read NFC tags. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/barcode-reader">barcode-reader</a> - A web component that reads barcodes via the Shape Detection API via a Web Worker. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/lite-tiktok">lite-tiktok</a> - A web component that lazy loads TikTok embeds. Currently experimental and a work in progress. Web Component (WC).</p>

<p><a href="https://github.com/justinribeiro/html5-dragdroptouch-shim ">html5-dragdroptouch-shim</a> - An opinionated shim that polyfills HTML5 drag and drop support on mobile devices with Event.ComposedPath() support.  Web Shim/Polyfill.</p>

<p><a href="https://github.com/justinribeiro/blog-pwa">blog-pwa</a> - Combining Lit Web Components, Hugo, Service Worker, and Rollup into a progressive web app blog. Runs this blog.</p>

<p><a href="https://github.com/justinribeiro/ginger">ginger</a> - The OG a WebGL morph target progressive web application demo.</p>

<p><a href="https://github.com/justinribeiro/onscreen-keyboard-html5-canvas-multitouch">onscreen-keyboard-html5-canvas-multitouch</a> - The OG Multitouch keyboard implemented with HTML5 canvas, touch events API and Magictouch.js</p>
