## Source Code and Software

Per a [recent post](/chronicle/2026/02/12/fork-to-local-my-ongoing-quest-away-from-cloud-services/), I've been in the process of moving my source code for a lot of projects to more controllable infrastructure (e.g., my own hardware). In the mean time, here's some of the stuff hosted in various places in various running states. Use as you must.

## Web Software

* [Depth of Field Calculator](https://dof-calculator.pwa.run/) - Because sometimes I just don't want to do the math when I'm in the field with my large format cameras. Progressive Web App (PWA).
* [Speedometer](https://speedometer.pwa.run/) - A tiny no-frills speedometer progressive web app based on Geolocation, AmbientLightSensor, and WakeLock API. Originally developed when my speedometer on our 2005 Toyota Prius was not working. Progressive Web App (PWA).
* [Reading Light](https://readinglight.pwa.run/) - A simple screen color changer for reading purposes. Originally developed so I could read by phone light to my twins. Progressive Web App (PWA).
* [lite-youtube](https://github.com/justinribeiro/lite-youtube) - The fastest little YouTube embedder this side of the internet for your website. Web Component (WC).
* [share-to-mastodon](https://github.com/justinribeiro/share-to-mastodon) - A small web component that uses a dialog to allow setting of a target Mastodon instance from the user to allow easier sharing. Web Component (WC).
* [code-block](https://github.com/justinribeiro/code-block) - A web component that displays colorfully formatted code with Prism.js and LitElement. Web Component (WC).
* [stl-part-viewer](https://github.com/justinribeiro/stl-part-viewer) - A lit-element web component that uses Three.js to display an STL model file. Web Component (WC).
* [toggle-tooltip](https://github.com/justinribeiro/toggle-tooltip) - An accessible, fast, and WCAG 2.1 compliant toggle style tooltip web component using role=status. Web Component (WC).
* [webnfc-reader](https://github.com/justinribeiro/webnfc-reader) - A lit-element web component that uses WebNFC to read NFC tags. Web Component (WC).
* [barcode-reader](https://github.com/justinribeiro/barcode-reader) - A web component that reads barcodes via the Shape Detection API via a Web Worker. Web Component (WC).
* [lite-tiktok](https://github.com/justinribeiro/lite-tiktok) - A web component that lazy loads TikTok embeds. Currently experimental and a work in progress. Web Component (WC).
* [html5-dragdroptouch-shim](https://github.com/justinribeiro/html5-dragdroptouch-shim) - An opinionated shim that polyfills HTML5 drag and drop support on mobile devices with Event.ComposedPath() support. Web Shim/Polyfill.
* [blog-pwa](https://github.com/justinribeiro/blog-pwa) - Combining Lit Web Components, Hugo, Service Worker, and Rollup into a progressive web app blog. Runs this blog.
* [ginger](https://github.com/justinribeiro/ginger) - The OG a WebGL morph target progressive web application demo.
* [onscreen-keyboard-html5-canvas-multitouch](https://github.com/justinribeiro/onscreen-keyboard-html5-canvas-multitouch) - The OG Multitouch keyboard implemented with HTML5 canvas, touch events API and Magictouch.js

## Client Software

* [zotero-google-scholar-citation-count](https://github.com/justinribeiro/zotero-google-scholar-citation-count) - Zotero plugin for fetching number of citations from Google Scholar.
* [android-hostcardemulation-sample](https://github.com/justinribeiro/android-hostcardemulation-sample) - Android HostCardEmulation sample that implements NFC Forum Type 4 Tag Operation.
* [rollup-plugin-asset-build-size-compare](https://github.com/justinribeiro/rollup-plugin-asset-build-size-compare) - Track asset build sizes and compare over time with Rollup. An opinionated and modified version of Wes's rollup-plugin-size, which was the Rollup port of Jason's webpack size-plugin.
* [dockerfiles](https://github.com/justinribeiro/dockerfiles) - Various Dockerfiles including chrome-headless, lighthouse, and other tooling.
* [devtools-to-video](https://github.com/justinribeiro/devtools-to-video) - Output a video file from screenshot frames within a Chrome DevTools JSON trace file.
* [ribeiro-social-sync.py](https://github.com/justinribeiro/ribeiro-social-sync.py) - Sync Mastodon to Twitter to meet friends where they are and save them from Nazi's. The software worked, but Twitter still fell.
* [wan-snmp-to-firebase-bridge](https://github.com/justinribeiro/wan-snmp-to-firebase-bridge) - Python script that uses Snimpy to extract SNMP data for WAN traffic and push to Firebase; originally for Google I/O and then Google fucked up my shit the night before and I had to fix it.
* [hce-to-chromenfc-app](https://github.com/justinribeiro/hce-to-chromenfc-app) - OG Chrome App that talks to Android host-based card emulation application.
* [ReplicatorG w/MQTT Status](https://github.com/justinribeiro/ReplicatorG) - OG Internet connected 3D printer; I talked about this build a lot, including at Eclipsecon back in the day.
* [idlemailcheck](https://github.com/justinribeiro/ReplicatorG) - IMAP IDLE mail check with Snarl notification for Perl.
